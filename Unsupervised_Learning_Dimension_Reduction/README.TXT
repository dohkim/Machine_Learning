source code Folder contains 2 data set(csv files) and 5 jupyter notebook files.

jupyter notebook file description
- clustering.ipynb : step 1,  Clustering (2 Clustering * 2 Datasets = 4 combination)
- dr.ipynb : step2, Dimension Rectuction (4 DR * 2 Datasets = 8 combination)
- cluster_w_dr.ipynb : step3,  Use DR then Clustering (4 DR * 2 Clustering * 2 Datasets = 16 combination)
- nn_w_dr.ipynb : Use Step 2 (DR) then NN (4 DR * 1 Dataset = 4 combination)
- nn_w_cluster.ipynb : Use Step 1 (Clustering) then NN (2 Clustering * 1 Dataset = 2 combination)

require package to run jupyter notebook files
- matplotlib
- seaborn
- pandas
- numpy
- sklearn
- scipy
- yellowbrick
